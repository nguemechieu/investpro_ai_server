InvestPro AI Server Documentation

Overview

The InvestPro AI Server is a gRPC-based backend service designed to provide real-time predictions based on market data using a deep learning model. It exposes prediction and health-check endpoints to integrate seamlessly with frontend clients or trading platforms.

Features

Serves predictions using a trained deep learning model (.h5 file)

Accepts 11 numerical market features as input

Returns prediction label (e.g., "up" or "down") and confidence score

Provides a health check endpoint for liveness monitoring

Uses gRPC for efficient and type-safe communication

predict.proto Definition

service Predictor {
  rpc Predict (MarketDataRequest) returns (PredictionResponse);
  rpc HealthCheck (google.protobuf.Empty) returns (HealthStatus);
}

Predict: Accepts 11 market features (e.g., open, close, rsi) and returns prediction and confidence.

HealthCheck: Returns simple string status (e.g., "SERVING").

Server Setup (server.py)

server = grpc.server(futures.ThreadPoolExecutor(max_workers=10))
predict_pb2_grpc.add_PredictorServicer_to_server(PredictorService(), server)
server.add_insecure_port("[::]:50051")
server.start()

Starts the server on port 50051

Loads the trained model if it exists

Optionally generates a dummy model if none is found

Running the Server

python server.py

Ensure you have:

Python 3.8+

TensorFlow

grpcio and grpcio-tools

Install dependencies:

pip install -r requirements.txt

Client Interaction (Java Example)
